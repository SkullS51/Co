# Internet Cowboy Security (ICO)

Zero-trust, zero-cloud, on-device security guard.
Detects phishing overlays, malicious notifications, accessibility abuse, and fake system prompts.

## Current state
- React + TypeScript frontend
- AccessibilityService for overlay detection
- PWA installable (manifest + icons needed)
- Bodyguard keyboard prototype

## Setup
```bash
npm install
npm run dev
